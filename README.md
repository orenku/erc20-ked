ERC20 Token + Escrow service

This project demonstrates a basic Hardhat use case. 
It creates an ERC20 token using openZeppelin ERC20 library extended to support mint and Burn, 

In addition it demonstrate escrow contract.

test are used to run and check the validity of the contracts. 

Escrow service is based on 'https://blog.logrocket.com/develop-solidity-smart-contracts-hardhat/'
